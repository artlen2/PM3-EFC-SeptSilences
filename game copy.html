Comment naviguer

Les objets disponibles

Téléphone Toucher le téléphone pour avancer dans la scène correspondante.
Café Toucher la tasse de café pour avancer dans la scène correspondante.
Bain Toucher l'objet du bain pour avancer dans la scène correspondante.
Liste Toucher la liste pour avancer dans la scène correspondante.

Chaque scène est liée à un objet précis. Seul l'objet mis en évidence en bas de l'écran est actif — toucher les autres n'aura aucun effet.

Suivez les indications pour savoir quel objet interagir. L'expérience avancera une fois l'objet touché.

Vous pouvez également appuyer sur la barre espace pour passer une scène.
